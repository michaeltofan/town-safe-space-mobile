# town-safe-space-mobile

Visual Flutter MVP for **TOWN** — a global civic app with local-only access.

Pilot community: **Italy · Milano · Brera**

## What this prototype includes

Welcome → Find Community → Verification → Access Granted → Local Feed → Story Detail → Local Discussion

- Warm editorial design (`#F8F3EA`)
- Vertical `PageView` Substack-style local story cards
- Mock data only (no GPS, payments, backend, auth)

## Run locally

```bash
flutter pub get
flutter run
```

On a connected Android/iOS device or emulator:

```bash
flutter run -d <device-id>
```

List devices:

```bash
flutter devices
```

## Analyze

```bash
flutter analyze
```
